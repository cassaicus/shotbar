Shotbar
Shotbar は、macOSのメニューバーから素早く、連続してスクリーンショットを撮影するために設計された軽量なユーティリティツールです。

主な特徴
メニューバー常駐: フルスクリーンのスクリーンショットを連続で保存。
キャプチャ機能: キャプチャ後、カーソルキーを操作をするので、表示中のアプリケーションのページを進めることができます。
